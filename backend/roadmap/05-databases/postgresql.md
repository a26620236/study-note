# PostgreSQL

## 簡介

PostgreSQL is an advanced, open-source RDBMS known for robustness, extensibility, and standards compliance. Supports complex queries, custom data types, full-text search, and ACID properties. Highly extensible with strong concurrency support, suitable for web apps to data warehousing.

**基礎教學（12個核心部分）**

| 類別 | 主要主題 |
|------|--------|
| **查詢數據** | SELECT、Column aliases、ORDER BY、SELECT DISTINCT |
| **數據篩選** | WHERE、AND/OR、LIMIT、FETCH、IN、BETWEEN、LIKE、IS NULL |
| **多表聯合** | INNER JOIN、LEFT JOIN、Self-join、FULL OUTER JOIN、CROSS JOIN |
| **數據分組** | GROUP BY、HAVING、Grouping Sets、Cube、Rollup |
| **集合操作** | UNION、INTERSECT、EXCEPT |
| **子查詢** | Subquery、Correlated Subquery、ANY、ALL、EXISTS |
| **通用表達式** | CTE、遞迴查詢 |
| **數據修改** | INSERT、UPDATE、DELETE、UPSERT |
| **事務處理** | BEGIN、COMMIT、ROLLBACK |
| **數據轉移** | CSV 匯入/匯出 |
| **表格管理** | CREATE、ALTER、DROP、TRUNCATE、複製表格 |
| **約束條件** | Primary Key、Foreign Key、CHECK、UNIQUE、NOT NULL、DEFAULT |

**數據類型**

- **數值類型**：Boolean、Integer（SMALLINT、INT、BIGINT）、NUMERIC、DOUBLE PRECISION、REAL
- **文本類型**：CHAR、VARCHAR、TEXT
- **時間類型**：DATE、Timestamp、TIME、Interval
- **進階類型**：UUID、Array、hstore、JSON、XML、BYTEA、Composite Types、Enum

**進階教學四大主題**
