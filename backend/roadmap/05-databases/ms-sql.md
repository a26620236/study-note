# MS SQL

## 簡介

Microsoft SQL Server is a relational database management system for structured data management. Supports querying, transactions, data warehousing with T-SQL, SSIS integration, SSAS analytics, and SSRS reporting. Used in enterprise environments for reliable storage and processing.

#### 1. Tutorials for SQL Server
> 原文：https://learn.microsoft.com/en-us/sql/sql-server/tutorials-for-sql-server-2016?view=sql-server-ver15

**教學概述**

Microsoft Learn 提供完整的 SQL Server 教學資源，幫助學習新技術和功能。早期版本的教學通常也適用於較新版本。

**主要教學分類**

| 技術領域 | 說明 |
|---------|------|
| **Database Engine Tutorials** | 學習如何連接資料庫引擎實例並建立和執行 Transact-SQL 語句 |
| **Enterprise Information Management (EIM) Tutorials** | 學習如何使用 Microsoft EIM 技術管理企業信息 |
| **Integration Services Tutorials** | 學習如何創建和部署 SSIS packages |
| **Replication tutorials** | 學習如何使用 SQL Server Management Studio 設置和運行複製拓撲 |
| **Reporting Services tutorials (SSRS)** | 學習如何創建基本報告和訂閱 |
| **R tutorials for SQL machine learning** | 學習如何使用 R 或 Python 處理 SQL Server 數據 |
