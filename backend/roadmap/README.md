## 📖 使用建議

### 學習路徑推薦

1. **入門階段**
   - 從 [01. Introduction & Internet 基礎](./01-introduction-internet/) 開始
   - 選擇一門 [02. Programming Languages](./02-programming-languages/)
   - 學習 [04. APIs](./04-apis/) 的基本概念

2. **進階階段**
   - 深入了解 [05. Databases](./05-databases/)
   - 學習 [06. Caching](./06-caching/) 和 [07. Web Servers](./07-web-servers/)
   - 掌握 [08. Authentication & Security](./08-authentication-security/)

3. **架構階段**
   - 研究 [10. Architectural Patterns](./10-architectural-patterns/)
   - 了解 [11. Message Brokers](./11-message-brokers/) 和 [12. Real-time Communication](./12-realtime-communication/)

4. **維運階段**
   - 學習 [13. Monitoring & Observability](./13-monitoring-observability/)
   - 掌握 [14. CI/CD & DevOps](./14-cicd-devops/)
   - 深入 [15. Scalability & Performance](./15-scalability-performance/)

### 如何使用這些資源

1. **主題式學習**：根據你目前的專案需求，選擇相關主題深入學習
2. **系統化學習**：按照上述學習路徑，循序漸進地完成所有主題
3. **查閱參考**：作為快速參考手冊，需要時查找特定主題的資源
