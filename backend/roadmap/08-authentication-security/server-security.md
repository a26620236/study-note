# Server Security

## 簡介

Server security protects servers from threats through patch management, access control, firewalls, encryption, security hardening, regular backups, and monitoring. Ensures confidentiality, integrity, and availability of data and services with continuous threat detection and response.

**什麼是伺服器強化（Server Hardening）**

根據 NIST，「強化（Hardening）」是修補漏洞或關閉不必要服務的過程。伺服器強化透過消除不必要的流程和關閉潛在入侵點來縮小組織的攻擊面。

**關鍵實踐**

**存取控制與身份驗證**
- 實施多因素驗證（MFA）
- 強制使用含字母、數字和特殊字符的複雜密碼
- 多次登入失敗後鎖定帳戶
- 限制伺服器的實體和數位存取

**軟體管理**
- 定期應用作業系統補丁
- 根據安全標準更新第三方軟體
- 移除未使用的應用程式和服務
- 禁用不必要的 USB 端口和協議

**資料保護**
- 部署防火牆和防毒解決方案
- 使用 AES 加密或自加密硬碟
- 建立完整的備份解決方案
- 建立並維護配置管理系統

**監控與維護**
- 持續監控伺服器存取和登入
- 定期審計防火牆規則
- 追蹤系統補丁和更新
- 維護詳細的日誌和時間戳

**主要優勢**

- 減少攻擊面和漏洞曝露
- 防止未授權存取關鍵基礎設施
- 降低勒索軟體和惡意軟體威脅
- 強化整體安全態勢
- 幫助組織符合合規標準（NIST、CIS Benchmarks）
