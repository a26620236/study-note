## 4.4 遞迴樹方法（續）

### 範例詳解：T(n) = 3T(n/4) + Θ(n²)

（續自 Part A）

#### 樹的結構分析

**樹的高度計算**：
- 第 i 層的子問題大小為 n/4ⁱ
- 當 n/4ⁱ = 1 時到達葉節點，即 i = log₄ n
- 因此樹有內部節點在深度 0, 1, 2, ..., log₄ n - 1
- 葉節點在深度 log₄ n

**各層的成本**：
- **深度 i 的節點數**：3ⁱ（每層是上層的 3 倍）
- **每個深度 i 的節點成本**：c(n/4ⁱ)²
- **深度 i 的總成本**：3ⁱ × c(n/4ⁱ)² = (3/16)ⁱ cn²
- **葉節點數量**：3^(log₄ n) = n^(log₄ 3)（使用公式 3.21）
- **葉節點總成本**：Θ(n^(log₄ 3))

#### 總成本計算

所有層的成本相加：
```
T(n) = cn² + (3/16)cn² + (3/16)²cn² + ... + (3/16)^(log₄ n)cn² + Θ(n^(log₄ 3))
```

這是一個**遞減的幾何級數**，使用幾何級數公式（方程式 A.7）：
```
Σ(i=0 to log₄ n) (3/16)ⁱ cn² < Σ(i=0 to ∞) (3/16)ⁱ cn²
                               = cn² / (1 - 3/16)
                               = (16/13)cn²
```

**最終結果**：
```
T(n) = (16/13)cn² + Θ(n^(log₄ 3))
```

由於 log₄ 3 ≈ 0.793 < 2，所以 n^(log₄ 3) = o(n²)

**結論**：T(n) = O(n²)

**關鍵洞察**：
- 成本從根到葉子呈**幾何遞減**
- **根節點的成本主導**整體成本
- 葉節點成本雖然有 n^(log₄ 3) 個，但每個只貢獻 Θ(1)，總和仍小於 n²

#### 替代法驗證

猜測：T(n) = O(n²)
假設：T(n) ≤ dn² 對某個常數 d > 0

**歸納證明**：
```
T(n) ≤ 3T(n/4) + cn²
     ≤ 3d(n/4)² + cn²
     = (3/16)dn² + cn²
     ≤ dn²
```

最後一步成立的條件：選擇 d ≥ (16/13)c

**基底情況**：選擇足夠大的 d 使得對於 1 ≤ n < n₀，有 dn² ≥ d ≥ T(n)

**結論**：T(n) = O(n²) 得證

### 不規則樹的範例：T(n) = T(n/3) + T(2n/3) + Θ(n)

**特性**：這個遞迴樹是**不平衡的**，不同的路徑有不同的長度。

#### 樹的結構（參考 Figure 4.2）

**子問題大小**：
- 向左走：子問題大小變為 1/3
- 向右走：子問題大小變為 2/3

**樹的高度**：
- **最短路徑**（一直向左）：當 (1/3)ʰ n < n₀ 時到達葉節點，高度約 log₃ n
- **最長路徑**（一直向右）：當 (2/3)ʰ n < n₀ 時到達葉節點，高度約 log₃/₂ n

精確地說，最長路徑的高度為 h = ⌊log₃/₂(n/n₀)⌋ + 1 = Θ(lg n)

#### 成本分析

**內部節點成本**：
- 每一層的成本總和 ≤ cn（因為所有節點大小總和不超過 n）
- 樹的高度 = Θ(lg n)
- **內部節點總成本** = O(n lg n)

**葉節點成本**：
初步想法：用完全二叉樹的葉節點數量來上界估計
- 完全二叉樹高度 h = ⌊log₃/₂ n⌋ + 1
- 葉節點數 = 2ʰ ≤ 2n^(log₃/₂ 2) = O(n^1.71)

但這個界太鬆了！實際上可以證明葉節點總數為 O(n)。

#### 計算葉節點數量

定義 L(n) 為遞迴樹中葉節點的數量，滿足遞迴：
```
L(n) = {  1                    若 n < n₀
       {  L(n/3) + L(2n/3)     若 n ≥ n₀
```

**替代法證明 L(n) = O(n)**：
假設 L(n) ≤ dn
```
L(n) = L(n/3) + L(2n/3)
     ≤ d(n/3) + d(2n/3)
     = dn
```

對任何 d > 0 都成立。選擇 d = 1 即可處理基底情況 L(n) = 1。

**葉節點總成本**：L(n) × Θ(1) = Θ(n)

#### 最終結論

```
T(n) = O(n lg n)  （內部節點）
     + Θ(n)       （葉節點）
     = O(n lg n)
```

**重要觀察**：
- 內部節點成本主導總成本（而非葉節點）
- 儘管樹不平衡，但透過仔細分析仍能得出緊密界限
- 可以用替代法證明 T(n) = Θ(n lg n)（見練習 4.4-3）

**方法論建議**：
- 遞迴樹可用於產生直覺和猜測
- 最好用替代法驗證遞迴樹得出的結果
- 對於複雜情況，可考慮使用更強大的數學工具（如主定理或 Akra-Bazzi 方法）

## 4.5 主定理（Master Method）

### 主定理的形式

主定理提供了一個「食譜式」的方法來解決以下形式的**主遞迴（Master Recurrence）**：

```
T(n) = aT(n/b) + f(n)
```

其中：
- **a > 0**：子問題的數量
- **b > 1**：子問題大小的縮減因子
- **f(n)**：**驅動函數（Driving Function）**，包含分割和合併的成本

**實際含義**：
- 將大小為 n 的問題分成 a 個子問題
- 每個子問題的大小為 n/b
- 分割和合併的總時間為 f(n)

**例子**：
- Strassen 演算法：a=7, b=2, f(n)=Θ(n²)
- Merge sort：a=2, b=2, f(n)=Θ(n)

### 關於向下取整和向上取整的處理

**重要簡化**：主定理允許你**忽略向下取整（floor）和向上取整（ceiling）**符號。

**例子**：
- 嚴格的 merge sort 遞迴：T(n) = T(⌊n/2⌋) + T(⌈n/2⌉) + Θ(n)
- 簡化後的主遞迴：T(n) = 2T(n/2) + Θ(n)
- 兩者的漸進解相同：Θ(n lg n)

**為何可以忽略？**
無論如何對引數進行取整，主定理提供的漸進界限保持不變。詳見 Section 4.7 的充分條件。

### 主定理（Theorem 4.1）

**定理陳述**：
設 a > 0, b > 1 為常數，f(n) 為定義在所有足夠大實數上的非負驅動函數。定義遞迴：
```
T(n) = aT(n/b) + f(n)
```

定義**分水嶺函數（Watershed Function）**：n^(logₐ b)

則 T(n) 的漸進行為可分為三種情況：

#### Case 1：驅動函數多項式地小於分水嶺函數

**條件**：存在常數 ε > 0 使得 f(n) = O(n^(logₐ b - ε))

**結論**：T(n) = Θ(n^(logₐ b))

**直覺**：
- 分水嶺函數比驅動函數成長得更快（至少快一個多項式因子 n^ε）
- 遞迴樹中，每層的成本從根到葉**幾何遞增**
- **葉節點成本主導**總成本

#### Case 2：驅動函數與分水嶺函數成長速度相近

**條件**：存在常數 k ≥ 0 使得 f(n) = Θ(n^(logₐ b) lg^k n)

**結論**：T(n) = Θ(n^(logₐ b) lg^(k+1) n)

**特例（最常見，k=0）**：
- 若 f(n) = Θ(n^(logₐ b))
- 則 T(n) = Θ(n^(logₐ b) lg n)

**直覺**：
- 驅動函數以 lg^k n 的因子比分水嶺函數成長得快
- 遞迴樹中，每層的成本大致相同（相差多對數因子）
- 樹的高度為 Θ(lg n)，因此解多了一個 lg n 因子

#### Case 3：驅動函數多項式地大於分水嶺函數

**條件**：
1. 存在常數 ε > 0 使得 f(n) = Ω(n^(logₐ b + ε))
2. **正則性條件（Regularity Condition）**：存在常數 c < 1 使得對所有足夠大的 n，af(n/b) ≤ cf(n)

**結論**：T(n) = Θ(f(n))

**直覺**：
- 驅動函數比分水嶺函數成長得更快（至少快一個多項式因子 n^ε）
- 遞迴樹中，每層的成本從根到葉**幾何遞減**
- **根節點成本主導**總成本

**正則性條件的含義**：
- 確保驅動函數在遞迴中的成長「穩定」
- 大多數多項式有界函數都滿足這個條件
- 有些函數可能在局部成長緩慢，但整體成長快速，這些函數可能不滿足正則性條件

### 使用主定理

使用主定理只需三步：
1. 識別 a, b, f(n)
2. 計算分水嶺函數 n^(logₐ b)
3. 判斷適用哪個情況並寫下答案

#### 範例 1：T(n) = 9T(n/3) + n

- a = 9, b = 3
- 分水嶺函數：n^(log₃ 9) = n² = Θ(n²)
- f(n) = n = O(n²⁻ᵋ)，對任何 ε ≤ 1
- **適用 Case 1**，ε = 1
- **解**：T(n) = Θ(n²)

#### 範例 2：T(n) = T(2n/3) + 1

- a = 1, b = 3/2
- 分水嶺函數：n^(log₃/₂ 1) = n⁰ = 1
- f(n) = 1 = Θ(n^(log₃/₂ 1) lg⁰ n) = Θ(1)
- **適用 Case 2**，k = 0
- **解**：T(n) = Θ(lg n)

#### 範例 3：T(n) = 3T(n/4) + n lg n

- a = 3, b = 4
- 分水嶺函數：n^(log₄ 3) ≈ n^0.793 = O(n^0.793)
- f(n) = n lg n = Ω(n^(log₄ 3 + ε))，可取 ε ≈ 0.2
- 檢查正則性條件：af(n/b) = 3(n/4)lg(n/4) ≤ (3/4)n lg n = cf(n)，c = 3/4 < 1 ✓
- **適用 Case 3**
- **解**：T(n) = Θ(n lg n)

#### 範例 4：T(n) = 2T(n/2) + n lg n

- a = 2, b = 2
- 分水嶺函數：n^(log₂ 2) = n
- f(n) = n lg n = Θ(n lg¹ n)
- **適用 Case 2**，k = 1
- **解**：T(n) = Θ(n lg² n)

### 應用於本章的遞迴

#### Merge Sort：T(n) = 2T(n/2) + Θ(n)

- a = 2, b = 2
- 分水嶺函數：n^(log₂ 2) = n
- f(n) = Θ(n)
- **適用 Case 2**，k = 0
- **解**：T(n) = Θ(n lg n)

#### 簡單矩陣乘法：T(n) = 8T(n/2) + Θ(1)

- a = 8, b = 2
- 分水嶺函數：n^(log₂ 8) = n³
- f(n) = Θ(1) = O(n³⁻ᵋ)，對任何 ε < 3
- **適用 Case 1**
- **解**：T(n) = Θ(n³)

#### Strassen 演算法：T(n) = 7T(n/2) + Θ(n²)

- a = 7, b = 2
- 分水嶺函數：n^(log₂ 7) ≈ n^2.807
- f(n) = Θ(n²) = O(n^(lg 7 - ε))，取 ε = 0.8
- **適用 Case 1**
- **解**：T(n) = Θ(n^(lg 7))

### 主定理不適用的情況

主定理並非萬能，有些情況無法使用：

#### 情況 1：驅動函數和分水嶺函數無法比較

某些函數（如振盪函數）無法與分水嶺函數進行漸進比較。實務上這種情況較少見。

#### 情況 2：落在間隙中

**Case 1 和 Case 2 之間的間隙**：
- f(n) = o(n^(logₐ b))，但分水嶺函數並未多項式地比驅動函數快

**Case 2 和 Case 3 之間的間隙**：
- f(n) = ω(n^(logₐ b))，驅動函數成長比分水嶺函數快，但不是多項式地快

**範例**：T(n) = 2T(n/2) + n/lg n
- a = 2, b = 2
- 分水嶺函數：n
- f(n) = n/lg n = o(n)（比 n 慢，但只慢了對數因子，非多項式）
- Case 1 不適用：因為 n/lg n ≠ O(n^(1-ε)) 對任何 ε > 0
- Case 2 不適用：因為 n/lg n ≠ Θ(n lg^k n)，k 必須 ≥ 0，但這裡 k = -1

**解決方法**：使用替代法或 Akra-Bazzi 方法（答案是 Θ(n lg lg n)）

#### 情況 3：正則性條件不滿足

Case 3 要求正則性條件 af(n/b) ≤ cf(n)，某些函數可能不滿足（見練習 4.5-5）。

**重要提醒**：儘管有這些限制，主定理仍能處理實務中絕大多數的分治遞迴。

## 4.6 連續主定理的證明

**注意**：本節包含數學證明，不需要理解證明也能應用主定理。如果只想學習如何使用主定理，可以跳過本節。

### 為何需要連續版本

**原始主定理（Theorem 4.1）**：
- 定義在整數上
- 需要處理向下取整和向上取整，技術細節複雜

**連續主定理（Continuous Master Theorem）**：
- 定義在實數上
- 不涉及向下取整和向上取整
- 證明包含理解主遞迴行為的主要思想
- 漸進解與原始主定理相同

### 證明結構概覽

證明分為三個部分：

1. **Lemma 4.2**：使用遞迴樹將簡化的主遞迴解歸結為求和式
2. **Lemma 4.3**：為求和式提供漸進界限，對應主定理的三種情況
3. **Theorem 4.4（連續主定理）**：將結果推廣到任意閾值常數 n₀

### Lemma 4.2：遞迴樹求和

**引理陳述**：
設 a > 0, b > 1 為常數，f(n) 為定義在實數 n ≥ 1 上的函數。則遞迴：
```
T(n) = {  Θ(1)           若 0 ≤ n < 1
       {  aT(n/b) + f(n) 若 n ≥ 1
```

有解：
```
T(n) = Θ(n^(logₐ b)) + Σ(j=0 to ⌊logₐ n⌋) aʲ f(n/bʲ)
```

**證明概要**（使用遞迴樹，參考 Figure 4.3）：

**內部節點**：
- 根的成本：f(n)
- 深度 1：a 個節點，每個成本 f(n/b)
- 深度 j：aʲ 個節點，每個成本 f(n/bʲ)
- 深度 j 的總成本：aʲ f(n/bʲ)

**葉節點**：
- 樹的高度：⌊logₐ n⌋ + 1
- 葉節點數量：a^(⌊logₐ n⌋ + 1) = Θ(n^(logₐ b))
- 每個葉節點成本：Θ(1)
- 葉節點總成本：Θ(n^(logₐ b))

**總成本**：
```
T(n) = [葉節點成本] + [內部節點成本]
     = Θ(n^(logₐ b)) + Σ(j=0 to ⌊logₐ n⌋) aʲ f(n/bʲ)
```

**三種情況的直覺**：
- **Case 1**：成本從根到葉幾何遞增，葉節點主導
- **Case 2**：每層成本大致相同（相差多對數因子）
- **Case 3**：成本從根到葉幾何遞減，根節點主導

### Lemma 4.3：求和式的漸進界限

**引理陳述**：
設 a > 0, b > 1 為常數，f(n) 為定義在實數 n ≥ 1 上的函數。則函數：
```
g(n) = Σ(j=0 to ⌊logₐ n⌋) aʲ f(n/bʲ)
```

的漸進行為可分為三種情況：

#### Case 1 證明：f(n) = O(n^(logₐ b - ε))

**目標**：證明 g(n) = O(n^(logₐ b))

**證明**：
```
g(n) = Σ(j=0 to ⌊logₐ n⌋) aʲ f(n/bʲ)
     = Σ(j=0 to ⌊logₐ n⌋) aʲ O((n/bʲ)^(logₐ b - ε))
     = O(n^(logₐ b - ε) Σ(j=0 to ⌊logₐ n⌋) (a/b^(logₐ b - ε))ʲ)
     = O(n^(logₐ b - ε) Σ(j=0 to ⌊logₐ n⌋) (b^ε)ʲ)
```

這是一個幾何級數，公比為 b^ε > 1，因此：
```
Σ(j=0 to ⌊logₐ n⌋) (b^ε)ʲ = O((b^ε)^(⌊logₐ n⌋ + 1))
                            = O(b^(ε(logₐ n + 1)))
                            = O(n^ε)
```

所以：
```
g(n) = O(n^(logₐ b - ε) × n^ε) = O(n^(logₐ b))
```

#### Case 2 證明：f(n) = Θ(n^(logₐ b) lg^k n)

**目標**：證明 g(n) = Θ(n^(logₐ b) lg^(k+1) n)

**證明概要**（詳見教科書）：
- 代入 f(n/bʲ) = Θ((n/bʲ)^(logₐ b) lg^k(n/bʲ))
- 簡化後得到：g(n) = Θ(n^(logₐ b) Σ(j=0 to ⌊logₐ n⌋) lg^k(n/bʲ))
- 求和式 Σ(j=0 to ⌊logₐ n⌋) (logₐ n - j)^k = Θ(log^(k+1)ₐ n)
- 結論：g(n) = Θ(n^(logₐ b) lg^(k+1) n)

**關鍵步驟**：
求和式可以上界為：
```
Σ(j=0 to ⌊logₐ n⌋) (logₐ n - j)^k ≤ Σ(j=1 to ⌊logₐ n⌋+1) j^k
                                   = O((⌊logₐ n⌋ + 1)^(k+1))
                                   = O(log^(k+1)ₐ n)
```

類似地可以證明下界為 Ω(log^(k+1)ₐ n)，因此求和式為 Θ(log^(k+1)ₐ n)。

#### Case 3 證明：正則性條件

**條件**：存在 0 < c < 1 使得對所有 n ≥ 1，af(n/b) ≤ cf(n)

**證明概要**：
正則性條件意味著驅動函數在遞迴中以幾何方式遞減。透過反覆應用這個條件，可以證明：
```
aʲ f(n/bʲ) ≤ cʲ f(n)
```

因此：
```
g(n) = Σ(j=0 to ⌊logₐ n⌋) aʲ f(n/bʲ)
     ≤ Σ(j=0 to ⌊logₐ n⌋) cʲ f(n)
     = f(n) Σ(j=0 to ∞) cʲ
     = f(n) × 1/(1-c)
     = O(f(n))
```

類似地可以證明 g(n) = Ω(f(n))，因此 g(n) = Θ(f(n))。

   - 三種情況的嚴格證明

### 關鍵洞察

1. **主定理的威力**：記住三種情況後，可以快速解決大多數分治遞迴
2. **分水嶺函數**：n^(logₐ b) 是判斷的關鍵，它代表遞迴樹的葉節點數量
3. **多項式分離**：Case 1 和 Case 3 都需要多項式因子 n^ε 的分離，對數因子不夠
4. **正則性條件**：Case 3 需要額外檢查，但大多數常見函數都滿足

### 實務建議

1. **優先使用主定理**：當遞迴符合主遞迴形式時，這是最快的方法
2. **當主定理不適用時**：
   - 使用遞迴樹產生猜測
   - 用替代法驗證
   - 考慮 Akra-Bazzi 方法（Part C 將介紹）

3. **檢查清單**：
   - □ 識別 a, b, f(n)
   - □ 計算 n^(logₐ b)
   - □ 比較 f(n) 和 n^(logₐ b)
   - □ 確認多項式分離（Case 1 或 Case 3）
   - □ 檢查正則性條件（僅 Case 3）
