## 4.6 連續主定理的證明（續）

### Lemma 4.3 Case 3 證明完成

**目標**：證明若存在 0 < c < 1 使得對所有 n ≥ 1，af(n/b) ≤ cf(n)，則 g(n) = Θ(f(n))

**證明**：

首先，因為 f(n) 出現在 g(n) 的定義式（當 j=0 時），且 g(n) 的所有項都是正數，所以 g(n) = Ω(f(n))。

現在證明 g(n) = O(f(n))。

**關鍵步驟**：反覆應用正則性條件 af(n/b) ≤ cf(n)，可得：
```
aʲ f(n/bʲ) ≤ cʲ f(n)
```

**代入求和式**：
```
g(n) = Σ(j=0 to ⌊logₐ n⌋) aʲ f(n/bʲ)
     ≤ Σ(j=0 to ⌊logₐ n⌋) cʲ f(n)
     ≤ f(n) Σ(j=0 to ∞) cʲ
     = f(n) × 1/(1-c)      （幾何級數，|c| < 1）
     = O(f(n))
```

結合上下界：g(n) = Θ(f(n)) ✓

**Lemma 4.3 證明完成**。

### Theorem 4.4：連續主定理

**定理陳述**：
設 a > 0, b > 1 為常數，f(n) 為定義在所有足夠大實數上的非負驅動函數。定義演算法遞迴：
```
T(n) = aT(n/b) + f(n)
```

則 T(n) 的漸進行為分為三種情況（與 Theorem 4.1 完全相同）：

1. **Case 1**：若存在 ε > 0 使得 f(n) = O(n^(logₐ b - ε))，則 T(n) = Θ(n^(logₐ b))
2. **Case 2**：若存在 k ≥ 0 使得 f(n) = Θ(n^(logₐ b) lg^k n)，則 T(n) = Θ(n^(logₐ b) lg^(k+1) n)
3. **Case 3**：若存在 ε > 0 使得 f(n) = Ω(n^(logₐ b + ε))，且 f(n) 滿足正則性條件 af(n/b) ≤ cf(n)（c < 1），則 T(n) = Θ(f(n))

**證明策略**：
將任意閾值常數 n₀ 的情況歸結到 Lemma 4.2（閾值為 1）的情況。

### 證明概要

**定義輔助函數**：
對於 n > 0，定義：
- T'(n) = T(n₀ n)
- f'(n) = f(n₀ n)

**轉換遞迴**：
```
T'(n) = T(n₀ n)
      = { Θ(1)           若 n₀ n < n₀（即 n < 1）
        { aT(n₀ n/b) + f(n₀ n)   若 n₀ n ≥ n₀（即 n ≥ 1）

      = { Θ(1)           若 n < 1
        { aT'(n/b) + f'(n)       若 n ≥ 1
```

這滿足 Lemma 4.2 的條件，因此：
```
T'(n) = Θ(n^(logₐ b)) + Σ(j=0 to ⌊logₐ n⌋) aʲ f'(n/bʲ)
```

### 各情況的證明

#### Case 1 證明

**已知**：f(n) = O(n^(logₐ b - ε))

**推導 f'(n)**：
```
f'(n) = f(n₀ n)
      = O((n₀ n)^(logₐ b - ε))
      = O(n^(logₐ b - ε))    （n₀ 是常數）
```

f'(n) 滿足 Lemma 4.3 的 Case 1 條件，因此求和式為 O(n^(logₐ b))。

**結論**：
```
T(n) = T'(n/n₀)
     = Θ((n/n₀)^(logₐ b)) + O((n/n₀)^(logₐ b))
     = Θ(n^(logₐ b)) + O(n^(logₐ b))
     = Θ(n^(logₐ b))        （Problem 3-5(b)）
```

#### Case 2 證明

**已知**：f(n) = Θ(n^(logₐ b) lg^k n)

**推導 f'(n)**：
```
f'(n) = f(n₀ n)
      = Θ((n₀ n)^(logₐ b) lg^k(n₀ n))
      = Θ(n^(logₐ b) lg^k n)    （消除常數項）
```

f'(n) 滿足 Lemma 4.3 的 Case 2 條件，因此求和式為 Θ(n^(logₐ b) lg^(k+1) n)。

**結論**：
```
T(n) = T'(n/n₀)
     = Θ((n/n₀)^(logₐ b)) + Θ((n/n₀)^(logₐ b) lg^(k+1)(n/n₀))
     = Θ(n^(logₐ b)) + Θ(n^(logₐ b) lg^(k+1) n)
     = Θ(n^(logₐ b) lg^(k+1) n)    （Problem 3-5(c)）
```

#### Case 3 證明

**已知**：
1. f(n) = Ω(n^(logₐ b + ε))
2. af(n/b) ≤ cf(n) 對所有 n ≥ n₀ 且某個 c < 1

**推導 f'(n)**：
```
f'(n) = f(n₀ n)
      = Ω((n₀ n)^(logₐ b + ε))
      = Ω(n^(logₐ b + ε))
```

**驗證正則性條件**：
對於 n ≥ 1（即 n₀ n ≥ n₀）：
```
af'(n/b) = af(n₀ n/b)
         ≤ cf(n₀ n)        （原始條件）
         = cf'(n)
```

f'(n) 滿足 Lemma 4.3 的 Case 3 條件，因此求和式為 Θ(f'(n))。

**結論**：
```
T(n) = T'(n/n₀)
     = Θ((n/n₀)^(logₐ b)) + Θ(f'(n/n₀))
     = Θ(f'(n/n₀))                （Case 3 中驅動函數主導）
     = Θ(f(n))
```

**Theorem 4.4 證明完成** ✓

## 4.7 Akra-Bazzi 遞迴

### 背景與動機

本節討論兩個進階主題：
1. 向下取整和向上取整的技術細節
2. Akra-Bazzi 方法：解決複雜分治遞迴的通用方法

### Akra-Bazzi 遞迴的定義

**一般形式**：
```
T(n) = f(n) + Σ(i=1 to k) aᵢ T(n/bᵢ)
```

其中：
- **k**：正整數（子問題的數量）
- **a₁, a₂, ..., aₖ ∈ ℝ**：嚴格正數（子問題的係數）
- **b₁, b₂, ..., bₖ ∈ ℝ**：嚴格大於 1（子問題大小的縮減因子）
- **f(n)**：驅動函數，定義在足夠大的非負實數上，且非負

### Akra-Bazzi 與主定理的比較

**主遞迴的限制**：
- 所有子問題大小相同（都是 n/b）
- 可以忽略向下取整和向上取整

**Akra-Bazzi 遞迴的優勢**：
- 可以處理**不同大小的子問題**（n/bᵢ 各不相同）
- 更通用
- 但需要額外條件來處理向下取整和向上取整

**例子**：
```
T(n) = T(n/3) + T(2n/3) + Θ(n)
```
這不是主遞迴（子問題大小不同），但是 Akra-Bazzi 遞迴。

## 向下取整和向上取整的處理

### 問題的根源

**為何需要小心？**
- 演算法通常處理整數大小的輸入
- 實數上的數學通常較簡單
- 但某些驅動函數可能表現「怪異」

**問題**：一般來說，在 Akra-Bazzi 遞迴中忽略向下取整和向上取整是**不安全的**。

**好消息**：大多數實務中遇到的驅動函數都「行為良好」，向下取整和向上取整不會影響漸進解。

### 多項式成長條件（Polynomial-Growth Condition）

**定義**：
函數 f(n) 定義在所有足夠大的正實數上，滿足**多項式成長條件**，若存在常數 y_n > 0 使得：

對於每個常數 α ≥ 1，存在常數 d > 1（依賴於 α）使得對所有 1 ≤ λ ≤ α 和 n ≥ y_n，有：
```
f(n)/d ≤ f(λn) ≤ df(n)
```

**直覺理解**：
- 一階近似：f(Θ(n)) = Θ(f(n))
- 函數在「適度」縮放下不會變化太大
- 實際上比 f(Θ(n)) = Θ(f(n)) 稍強一些

**重要性質**：
- 多項式成長條件蘊含 f(n) 是**漸進正數**（見練習 4.7-3）

### 滿足多項式成長條件的函數

**常見例子**：
任何形如 f(n) = Θ(n^α lg^β n lg lg^γ n) 的函數，其中 α, β, γ 為常數。

**不滿足的例子**：
- **指數函數**：f(n) = 2^n（見練習 4.7-2）
- **超指數函數**
- 某些多項式有界函數也可能不滿足

**實務意義**：
本書中使用的大多數多項式有界函數都滿足這個條件。

## Theorem 4.5：向下取整和向上取整的不變性

**定理陳述**：
設 T(n) 為定義在非負實數上的函數，滿足遞迴：
```
T(n) = f(n) + Σ(i=1 to k) aᵢ T(n/bᵢ)
```
其中 f(n) 滿足多項式成長條件。

設 T'(n) 為定義在自然數上的另一個函數，也滿足相同遞迴，除了每個 T(n/bᵢ) 被替換為 T(⌈n/bᵢ⌉) 或 T(⌊n/bᵢ⌋)。

**結論**：T'(n) = Θ(T(n))

### 定理的意義

**向下取整和向上取整的影響**：
- 它們代表遞迴引數的「微小擾動」
- 根據不等式（3.2），最多擾動 ±1
- 定理說明這種擾動不影響漸進解

**更大的擾動也能容忍**：
只要驅動函數 f(n) 滿足多項式成長條件，將任何項 T(n/bᵢ) 替換為 T(n/bᵢ + hᵢ(n))，其中：
```
|hᵢ(n)| = O(n / lg^(1+ε) n)
```
對某個常數 ε > 0 且足夠大的 n，漸進解仍然不受影響。

**實務啟示**：
分治演算法的分割步驟可以適度粗糙，而不影響執行時間遞迴的解。

## Akra-Bazzi 方法

### 方法概述

Akra-Bazzi 方法用於解決 Akra-Bazzi 遞迴（4.22）。根據 Theorem 4.5，該方法在向下取整和向上取整存在時仍然適用（只要驅動函數滿足多項式成長條件）。

### 求解步驟

**步驟 1：找出 p 值**
首先確定唯一的實數 p 使得：
```
Σ(i=1 to k) aᵢ / bᵢ^p = 1
```

**為何 p 存在且唯一？**
- 當 p → -∞ 時，和趨向 ∞
- 當 p 增加時，和遞減
- 當 p → ∞ 時，和趨向 0
- 因此存在唯一的 p 使得和等於 1

**步驟 2：計算解**
使用公式：
```
T(n) = Θ(n^p (1 + ∫₁ⁿ f(x)/x^(p+1) dx))
```

### 範例：T(n) = T(n/5) + T(7n/10) + n

**識別參數**：
- k = 2
- a₁ = a₂ = 1
- b₁ = 5, b₂ = 10/7
- f(n) = n

**步驟 1：求 p**
解方程：
```
(1/5)^p + (7/10)^p = 1
```

**數值估計**：
- p = 0: (1/5)⁰ + (7/10)⁰ = 1 + 1 = 2 > 1
- p = 1: (1/5)¹ + (7/10)¹ = 1/5 + 7/10 = 9/10 < 1
- 因此 0 < p < 1

**步驟 2：計算積分**
```
T(n) = Θ(n^p (1 + ∫₁ⁿ f(x)/x^(p+1) dx))
     = Θ(n^p (1 + ∫₁ⁿ x/x^(p+1) dx))
     = Θ(n^p (1 + ∫₁ⁿ x^(-p) dx))
```

使用微積分公式 ∫ x^k dx = x^(k+1)/(k+1)（k ≠ -1）：
```
∫₁ⁿ x^(-p) dx = [x^(1-p)/(1-p)]₁ⁿ
               = (n^(1-p) - 1)/(1-p)
```

由於 0 < p < 1，所以 1-p > 0 是正常數：
```
T(n) = Θ(n^p (1 + Θ(n^(1-p))))
     = Θ(n^p × Θ(n^(1-p)))      （因為 1-p > 0）
     = Θ(n^p × n^(1-p))
     = Θ(n)                      （Problem 3-5(d)）
```

**結論**：T(n) = Θ(n)

**注意**：我們不需要知道 p 的精確值（p ≈ 0.83978），只需知道 0 < p < 1 即可得出結論。

### Akra-Bazzi 方法 vs 主定理

| 特性 | 主定理 | Akra-Bazzi 方法 |
|------|--------|----------------|
| **適用範圍** | 子問題大小相同 | 子問題大小可不同 |
| **易用性** | 簡單（記住三種情況） | 需要微積分和推理 |
| **向下/上取整** | 可忽略 | 需要多項式成長條件 |
| **通用性** | 較限制 | 更通用 |
| **實務選擇** | 優先使用（當適用時） | 主定理不適用時的備選 |

**建議**：
- 當遞迴符合主遞迴形式時，優先使用主定理（更簡單）
- 當子問題大小不同或主定理不適用時，使用 Akra-Bazzi 方法
- 兩者都是演算法工具箱中的好工具

### Problem 4-1：遞迴範例

為以下每個遞迴給出漸進緊密的上下界：

**(a)** T(n) = 2T(n/2) + n³
- a=2, b=2, n^(logₐ b) = n
- f(n) = n³ = Ω(n^(1+ε)), ε=2
- 檢查正則性：2(n/2)³ = n³/4 ≤ cf(n), c=1/4 ✓
- **Case 3**: T(n) = Θ(n³)

**(b)** T(n) = T(8n/11) + n
- a=1, b=11/8, n^(logₐ b) = n⁰ = 1
- f(n) = n = Ω(1^(1+ε)), ε=1
- **Case 3**: T(n) = Θ(n)

**(c)** T(n) = 16T(n/4) + n²
- a=16, b=4, n^(logₐ b) = n²
- f(n) = n² = Θ(n²)
- **Case 2**, k=0: T(n) = Θ(n² lg n)

**(d)** T(n) = 4T(n/2) + n² lg n
- a=4, b=2, n^(logₐ b) = n²
- f(n) = n² lg n = Θ(n² lg¹ n)
- **Case 2**, k=1: T(n) = Θ(n² lg² n)

**(e)** T(n) = 8T(n/3) + n²
- a=8, b=3, n^(logₐ b) ≈ n^1.89
- f(n) = n² = Ω(n^(1.89+ε)), ε≈0.11
- **Case 3**: T(n) = Θ(n²)

**(f)** T(n) = 7T(n/2) + n² lg n
- a=7, b=2, n^(logₐ b) ≈ n^2.807
- f(n) = n² lg n = O(n^(2.807-ε)), ε≈0.8
- **Case 1**: T(n) = Θ(n^(lg 7))

**(g)** T(n) = 2T(n/4) + √n
- a=2, b=4, n^(logₐ b) = n^(1/2) = √n
- f(n) = √n = Θ(√n)
- **Case 2**, k=0: T(n) = Θ(√n lg n)

**(h)** T(n) = T(n-2) + n²
- 不是分治形式，是減法遞迴
- 展開：n² + (n-2)² + (n-4)² + ... + 1²
- 約有 n/2 項，每項最大 n²
- T(n) = Θ(n³)

### Problem 4-3：變數替換解遞迴

**問題**：解 T(n) = 2T(√n) + Θ(lg n)

**步驟**：

**(a)** 定義 m = lg n 和 S(m) = T(2^m)

代入：
```
T(n) = 2T(√n) + Θ(lg n)
T(2^m) = 2T(2^(m/2)) + Θ(m)
S(m) = 2S(m/2) + Θ(m)
```

**(b)** 解 S(m)
- 這是標準主遞迴，a=2, b=2, f(m)=Θ(m)
- **Case 2**: S(m) = Θ(m lg m)

**(c)** 轉換回 T(n)
```
T(n) = S(lg n) = Θ(lg n × lg lg n)
```

**結論**：T(n) = Θ(lg n lg lg n)

**(d)** 遞迴樹直覺
- 第一層：問題大小 n，成本 lg n
- 第二層：2 個大小 √n 的問題，總成本 2 × lg√n = lg n
- 第三層：4 個大小 n^(1/4) 的問題，總成本 4 × lg n^(1/4) = lg n
- ...
- 每層成本都是 lg n
- 樹的高度：lg lg n（因為 n → √n → n^(1/4) → ... → 2）
- 總成本：lg n × lg lg n

### Problem 4-5：Fibonacci 數列（生成函數）

**定義**：F₀ = 0, F₁ = 1, Fₙ = Fₙ₋₁ + Fₙ₋₂

**生成函數**：F(z) = Σ(i=0 to ∞) Fᵢ z^i

**(a)** 證明：F(z) = z + zF(z) + z²F(z)

**(b)** 解出：F(z) = z/(1-z-z²) = 1/√5 × (1/(1-φz) - 1/(1-φ̂z))
其中 φ = (1+√5)/2 是黃金比例，φ̂ = (1-√5)/2 是其共軛

**(c)** 展開：F(z) = Σ(i=0 to ∞) (1/√5)(φ^i - φ̂^i)z^i

**(d)** 結論：Fᵢ = φ^i/√5（四捨五入到最近整數）

因為 |φ̂| < 1，所以 φ̂^i/√5 < 1/2 對所有 i > 0。

**(e)** 證明：Fᵢ₊₂ ≥ φ^i 對所有 i ≥ 0

   - **主定理**（Master Method）：最實用的工具
   - **Akra-Bazzi 方法**：最通用但需要微積分

3. **數學基礎**（Part B & C）
   - 連續主定理的證明
   - 多項式成長條件
   - 向下取整和向上取整的處理

### 核心洞察

1. **分治法的威力**
   - Strassen 演算法證明了創新的分治策略可以突破傳統界限
   - 減少遞迴分支（從 8 到 7）比減少每層工作量更重要

2. **主定理的實用性**
   - 三種情況涵蓋大多數實務遞迴
   - 分水嶺函數 n^(logₐ b) 是判斷的關鍵
   - 需要多項式分離，對數分離不夠

3. **數學工具的層次**
   ```
   簡單 ←─────────────────────→ 複雜
   主定理  →  遞迴樹  →  替代法  →  Akra-Bazzi
   ↓           ↓          ↓          ↓
   快速       直覺      通用      最通用
   ```

4. **理論與實務的平衡**
   - 可以忽略向下取整和向上取整（在適當條件下）
   - 嚴格的數學證明確保我們可以在實務中保持「輕鬆但嚴謹」

### 方法選擇指南

**決策樹**：
```
遞迴是否符合 T(n) = aT(n/b) + f(n) 且子問題大小相同？
│
├─ 是 → 使用主定理
│       │
│       ├─ 能判斷情況 → 直接寫答案 ✓
│       └─ 落在間隙 → 用替代法或遞迴樹
│
└─ 否 → 子問題大小不同？
        │
        ├─ 是 → 考慮 Akra-Bazzi 方法
        │       （需要微積分）
        │
        └─ 否 → 使用替代法
                │
                ├─ 有好的猜測 → 證明之 ✓
                └─ 沒有猜測 → 用遞迴樹產生猜測
```

### 實務建議

1. **優先順序**：主定理 > 遞迴樹 > 替代法 > Akra-Bazzi
2. **組合使用**：用遞迴樹產生猜測，用替代法驗證
3. **檢查清單**：
   - □ 識別 a, b, f(n)
   - □ 計算 n^(logₐ b)
   - □ 比較 f(n) 和 n^(logₐ b)
   - □ 確認多項式分離（Case 1 或 3）
   - □ 檢查正則性條件（僅 Case 3）
