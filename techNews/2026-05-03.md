  - Pairs naturally with CSS modules and the new `with { type: 'css' }` import attribute.
- **Details**
  - Now well-supported across Chromium, Firefox, and Safari.
-------
- **核心**
  - `new CSSStyleSheet()` + `adoptedStyleSheets` 讓單一已解析 stylesheet 掛到多個 shadow root，不重複解析。
  - 取代以往「每個 web component 實例都塞一份 `<style>`」的舊作法。
- **影響**
  - 對發布 web component 的 design system 是實打實的優化——parse 時間與記憶體都看得到下降。
  - 自然搭配 CSS modules 與新的 `with { type: 'css' }` import attribute。

## Glossary

- **MCP (Model Context Protocol)**
  - An open protocol that standardizes how LLM agents connect to external tools, data, and services through a server interface.
  - 標準化 LLM agent 透過 server 介面去存取外部工具、資料與服務的開放協議。
- **Agent Memory**
  - Persistent storage that survives across an agent's sessions or restarts, holding goals, history, or learned preferences.
  - Agent 跨 session 或重啟後仍能保留的儲存層，存放目標、歷史與已學到的偏好。
- **Test Tags (Vitest)**
  - Metadata labels you attach to tests (e.g. `@slow`, `@network`) so you can filter, group, or skip them without restructuring files.
  - 你掛在測試上的 metadata 標籤（如 `@slow`、`@network`），用來篩選、分組或略過特定測試，不必重組檔案結構。
- **Constructable Stylesheets**
  - A web platform API (`new CSSStyleSheet()` + `adoptedStyleSheets`) that lets one parsed stylesheet be shared across many shadow roots.
  - Web 平台 API（`new CSSStyleSheet()` + `adoptedStyleSheets`），讓單一已解析 stylesheet 可以被多個 shadow root 共用。
- **Subgrid**
  - A CSS Grid feature where a child grid aligns its columns/rows to its parent grid's tracks instead of defining its own.
  - CSS Grid 功能：子 grid 對齊父 grid 的 column / row track，而不是各自定義。
- **Baseline (Web Platform)**
  - A status indicating a feature is interoperably supported across Chrome, Edge, Firefox, and Safari and considered safe to use.
  - 表示某項功能在 Chrome、Edge、Firefox、Safari 之間互通並可安心使用的狀態標籤。
- **Lakehouse / Iceberg / DuckLake**
  - Table formats that put database-like semantics (schema, transactions, time travel) on top of object-storage Parquet files.
  - 在物件儲存 Parquet 檔之上加上類資料庫語意（schema、交易、時光回溯）的 table 格式。
- **Schema Registry**
  - A central service that stores and versions message schemas (Avro / Protobuf / JSON Schema) used by Kafka producers and consumers.
  - 集中儲存與版本管理 Kafka 生產 / 消費端使用的 message schema（Avro / Protobuf / JSON Schema）的服務。
