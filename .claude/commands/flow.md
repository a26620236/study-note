使用 AskUserQuestion 工具，**一次呼叫**同時顯示以下三個問題（將三個問題全部放進同一個 questions 陣列）：

1. 檔名是什麼？（不需要附副檔名，我會自動加 `.md`）
2. 檔案要放在哪個目錄？（可以參考目前專案結構給建議）
3. flow 是什麼？請用 `=>` 分隔每個步驟，例如：`理解既有架構所有細節 => 擬定實作計畫(2-3 種方案並分析好壞) => 遵循團隊既有規範實作`

---

收到回答後，依照以下規則建立檔案：

**解析 flow：**
- 以 `=>` 切割，每段 trim 空白，得到 N 個步驟

**Mermaid flowchart：**
- 使用 `flowchart LR`
- 節點標籤必須簡短（中文 6 字以內），不加雙引號，格式：`A[短標籤]`
- 完整步驟說明放在下方「流程細節」section，不放在節點上
- 顏色用 `classDef` 定義並用 `class` 指派，依序循環：`#93c5fd`、`#6ee7b7`、`#fcd34d`、`#c4b5fd`、`#f9a8d4`，超過 5 個從頭循環
- `stroke:none,color:#000`

**檔案格式：**

```
# <檔名>

---

## Flowchart

\`\`\`mermaid
flowchart LR
    ...
\`\`\`

---

## 流程細節

### 1. <步驟一>

<步驟一的說明，從 flow 文字中提取，若無額外說明則留空一行>

---

### 2. <步驟二>

...依步驟數量產生，最後一個步驟後不加 ---
```

建立完成後告知檔案路徑，不需要其他說明。
