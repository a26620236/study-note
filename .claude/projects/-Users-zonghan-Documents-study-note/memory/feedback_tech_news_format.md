---
name: Tech News 雙語格式偏好
description: 科技新聞摘要的雙語排版規則：區塊對照而非逐行交替
type: feedback
---

在每個內容區塊內，英文全部放前面，中文全部放後面，中間空一行分隔。
不是一行英文接一行中文（交替格式），而是整個語言區塊集中在一起。

**Why:** 逐行交替（EN/ZH/EN/ZH）讀起來費力，閱讀節奏被打斷。

**How to apply:**

- **TL;DR**：所有英文 bullet 先列完，空一行，再列所有中文 bullet
- **新聞內文**（Core / Impact / Details）：先列全部英文 bullet，空一行，再列全部中文 bullet
- **Glossary**：每個詞條內部，英文定義先，空一行，再接中文定義
- **例外**：標題（`###` 雙行）維持一行英文一行中文（因為這樣才能雙語顯示標題）
